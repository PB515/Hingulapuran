# Parashurama story: full-frame scene art (v3, copy-paste prompts)

The Parashurama section is a **vertical hard-cut reel**: full-frame scenes that cut downward as you scroll, the campaign building to a held climax. So each beat is **one complete full-frame painting** (no layers, simpler than the Hingul scroll) with its own four-sided block-print border, like a framed cloth. **7 scenes**, faithful to the grantha (ch 76–87), ending on the Brahmakshatriya (no "sword at her feet", that is not this book's account).

Each image below is **one complete prompt in a code block** (full style baked in, copy, paste, generate). Above each: its **save filename**, **size**, **background**, and **"Used as."**

**Rules:**
- **Fill the frame.** Each scene is a complete 16:9 plate with a four-sided block-print border. The caption is overlaid by the site in the lower-left, so keep the lower-left of the composition calm.
- Opaque, no transparency needed. After saving: `node scripts/to-webp.mjs`.
- Keep one approved scene as a **style reference** so all 7 stay on-model.

### Negative prompt (set once, reuse for all)
```
no photorealism, no 3d render, no soft pastel, no cute, no chibi, no disney style, no modern flat vector, no corporate illustration, no gradients, no glow, no neon, no blur, no embedded text, no letters, no signature, no watermark, no extra fingers, no melted faces
```
