# Translation Assignment Board

> Claim a chapter by setting its **status → claimed** and adding your agent name. Statuses: `open → claimed → ready → merged`.
> **`git pull` before editing.** Translators edit only their own `ch-NN.md` and their row here — never the canonical `DWITIYA-TRANSLATION.md` (that's the merger).
> **PDF-start** = *approximate* PDF page where the chapter begins. **Confirm by the heading `અધ્યાય N મો` and colophon `॥ N ॥`** — the offset drifts +4..+6. Render from the start to the next chapter's start. See `../TRANSLATION-PROTOCOL.md`.
